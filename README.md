# EraMap 🗺️

A historical events explorer for iOS. Visualize Viking raids, Ottoman conquests, WWII battles, Roman Empire campaigns and the Turkish War of Independence on an interactive map with a timeline slider.

## Features

- 🗺️ Interactive MapKit map with historical event annotations
- ⏱️ Timeline slider — scrub through history from 753 BC to 1945
- ⚔️ 5 historical eras with color-coded pins
- 🔍 Tap any event to zoom in and read details
- ▶️ Auto-play mode — watch history unfold
- 🌙 Splash screen with custom app icon

## Eras

| Era | Period |
|-----|--------|
| ⚔️ Vikings | 793 – 1066 |
| 🌙 Ottoman Empire | 1299 – 1683 |
| 💀 World War II | 1939 – 1945 |
| 🏛️ Roman Empire | 753 BC – 476 AD |
| 🌙⭐ WWI & Turkish Independence | 1914 – 1938 |
